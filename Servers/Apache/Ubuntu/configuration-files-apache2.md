# Configuration Files in Apache2

## :open_file_folder: File's Structure 

Apache2 is configured by directives in configuration files. The standard directory structure is below:

```
etc
 |__apache2
      |__apache2.conf
      |__conf-available
      |    |__...
      |__conf-enabled
      |    |__...
      |__envvars
      |__magic
      |__mods-available
      |    |__...
      |__mods-enabled
      |    |__...
      |__ports.conf
      |__sites-available
      |    |__...
      |__sites-enabled
           |__...
```
